# Artifact Manifest

This document records what is preserved in the repository and what is generated locally. It exists so the paper artifact can be reviewed without accidentally deleting or regenerating expensive outputs.

## Link Stability

The folder name is intentionally unchanged:

```text
DiabeticRetinopathy_XAI/
```

This preserves the paper-facing GitHub link:

```text
https://github.com/DiellzaPervetica/Diabetic-Retinopathy-/tree/main/DiabeticRetinopathy_XAI
```

## Committed Artifact Files

| Path | Role |
|---|---|
| `README.md` | IEEE-style artifact overview, installation, workflow, data availability, results, limitations, and citation. |
| `CITATION.cff` | Machine-readable citation metadata for GitHub and archival tools. |
| `requirements.txt` | Python runtime package list. |
| `src/` | Reproducible source code for the full pipeline. |
| `reports/README.md` | Concise report manifest and manuscript-consistent metrics. |
| `reports/figures/` | Paper-facing figures preserved for inspection without rerunning training. |
| `data/**/.gitkeep` | Expected dataset and split folder placeholders. |
| `models/.gitkeep` | Expected local model-output folder placeholder. |
| `notebooks/.gitkeep` | Reserved notebook folder placeholder. |

## Generated Locally

These files or folders may be created by the pipeline and are intentionally ignored by git:

| Path | Generated by | Reason not committed |
|---|---|---|
| `data/raw/aptos2019/train_images/` | Dataset download/manual placement | Large external dataset; governed by Kaggle terms. |
| `data/processed/fundus_224/` | `src/08_preprocess_fundus_images.py` | Large derived image set. |
| `data/splits/*.csv` | `src/03_create_splits.py` | Reproducible split outputs. |
| `models/*.keras` | Training scripts | Large model checkpoints. |
| `outputs/figures/` | EDA/evaluation/Grad-CAM scripts | Full generated figure output. |
| `outputs/metrics/` | Evaluation scripts | Generated metric JSON/CSV outputs. |
| `outputs/gradcam/` | `src/07_gradcam_xai.py` | Generated explanation overlays. |
| `outputs/logs/` | Runtime scripts | Runtime logs. |

|---|---|

| Installation instructions | `README.md` |

| Preserved result figures | `reports/figures/` |

## Preservation Policy

- Do not rename `DiabeticRetinopathy_XAI`.
- Do not delete `reports/figures/`; these are the inspectable paper-facing results.
- Do not commit raw APTOS images or large model checkpoints unless the authors intentionally create a separate release artifact.
- If a formal DOI release is prepared, archive the exact git commit, add a frozen dependency file from the execution environment, and attach any non-recomputable generated outputs through an appropriate artifact repository.
