# Diabetic Retinopathy XAI Research Artifact

This repository contains the code artifact for the diabetic retinopathy grading study:

**Eyes as Windows to Health: Explainable EfficientNetB0 for Lightweight Five-Class Diabetic Retinopathy Grading in Fundus Images with Clinical Interpretability Support**

Primary artifact folder:

https://github.com/DiellzaPervetica/Diabetic-Retinopathy-/tree/main/DiabeticRetinopathy_XAI

The folder name `DiabeticRetinopathy_XAI` is intentionally preserved so existing paper and repository links remain valid.

## Artifact Contents

- Source code for dataset checking, EDA, preprocessing, model training, evaluation, cross-validation, and Grad-CAM.
- IEEE-style artifact README inside `DiabeticRetinopathy_XAI/`.
- Paper-facing figures under `DiabeticRetinopathy_XAI/reports/figures/`.
- Paper-facing figure documentation under `DiabeticRetinopathy_XAI/reports/`.
- Citation metadata under `DiabeticRetinopathy_XAI/CITATION.cff`.

Large generated files such as raw data, processed images, model checkpoints, and full run outputs are intentionally not committed. They are generated locally by the pipeline and ignored by git.
