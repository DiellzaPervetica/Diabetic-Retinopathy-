# Diabetic Retinopathy XAI

Ky repository përmban projektin universitar:

**Lightweight Deep Learning and Explainable AI for Diabetic Retinopathy Grading Using APTOS 2019 Fundus Images**

Projekti i plotë ndodhet në folderin:

```text
DiabeticRetinopathy_XAI/
```

README i detajuar, metodologjia, rezultatet, skriptet dhe figurat kryesore janë këtu:

[DiabeticRetinopathy_XAI/README.md](DiabeticRetinopathy_XAI/README.md)

Projekti përfshin baseline CNN, EfficientNetB0 me transfer learning, preprocessing për imazhe fundus, trajtim të class imbalance, formulim ordinal, cross-validation dhe Grad-CAM për Explainable AI.
